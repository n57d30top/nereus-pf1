# PF-1 Candidate Materials Table

## Purpose

This table translates the broader PF-1 material hypotheses into three more concrete starting candidates that a professor could use for first-pass downselection.

These are still **candidate concepts**, not finalized formulations, recipes, or production-ready materials.

## Candidate 1

### Working Name

PF1-C1 Hybrid bead with selective capture shell

### Candidate Class

Hybrid sorbent bead or granule

### Intended Structure

A mechanically stable bead or granule scaffold with a PFAS-affine outer capture zone and a regeneration-tolerant inner support phase.

### Why It Is A Strong Start

- best match to the current PF-1 cartridge and skid logic
- good chance of balancing packing behavior with selective capture
- easiest first translation from internal module thinking into real packed media

### Main Variables

- scaffold identity
- shell loading and shell thickness
- particle-size window
- surface accessibility
- regeneration tolerance

### Main Risk

Hydraulics may look strong while working PFAS capacity stays too low.

### Current Rank

`Priority 1`

### Companion Artifact

See `handover/PF1_C1_EXPLORATORY_SYNTHESIS_SOP.md`.

## Candidate 2

### Working Name

PF1-C2 Porous polymer sorbent with tunable capture motif

### Candidate Class

Porous functional polymer sorbent

### Intended Structure

A porous polymeric sorbent with a tunable PFAS-affine motif embedded in a mechanically bounded porous network.

### Why It Is A Strong Start

- strongest chemistry-tuning freedom
- good route for exploring capture/regeneration balance
- useful as the main chemistry-learning branch even if it does not become the final product

### Main Variables

- pore regime
- capture-motif density
- hydrophobic versus ionic balance
- binder fraction
- structural rigidity

### Main Risk

Capture may be strong at first pass while fouling or regeneration collapses.

### Current Rank

`Priority 2`

## Candidate 3

### Working Name

PF1-C3 Structured monolith with PFAS-affine coating

### Candidate Class

Structured adsorptive monolith or coated substrate

### Intended Structure

A structured flow support carrying a PFAS-affine active coating to reduce pressure penalty and simplify flow interpretation.

### Why It Is Worth Keeping

- strongest hydraulic story
- easiest candidate for interpreting pressure-drop behavior cleanly
- useful contrast candidate against the packed-media paths

### Main Variables

- substrate type
- coating identity
- coating thickness
- channel geometry
- regeneration compatibility

### Main Risk

Pressure behavior may improve while useful total capacity becomes too small.

### Current Rank

`Priority 3`

### Companion Artifact

See `handover/PF1_X1_CRYSTALLINE_BRANCH.md` and `structures/PF1_X1_HYPOTHETICAL_ACTIVE_PHASE.cif`.

## Downselection Logic

If only one candidate should be instantiated first, start with `PF1-C1`.

If two candidates should be instantiated first, start with:

1. `PF1-C1`
2. `PF1-C2`

Keep `PF1-C3` as the hydraulic-contrast branch unless there is a strong reason to prioritize pressure behavior above total useful capacity.

## What This Table Still Does Not Provide

- no exact molecular formula
- no exact precursor list
- no synthesis SOP
- no claim that any one candidate is already the winning real material

This table is meant to help a professor choose where to begin, not to pretend that the final PF-1 material has already been solved.
