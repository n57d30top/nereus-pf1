# PF1-C1 Exploratory Synthesis SOP

## Status

This is an **exploratory synthesis SOP** for the current top PF-1 start candidate, not a validated manufacturing protocol.

It is intended to help a professor instantiate a first bounded PF1-C1 screening material.

## Candidate Definition

`PF1-C1` is the current `Priority 1` PF-1 start candidate:

- hybrid sorbent bead or granule
- mechanically stable packed media
- selective PFAS-affine outer capture shell
- regeneration-tolerant support phase

## Intended Prototype Architecture

Use one bounded bead-style prototype with:

- porous bead or granule core
- thin selective outer shell
- regeneration-tolerant interfacial tie layer
- cartridge-friendly particle-size window

The conceptual formula is:

`core scaffold + tie layer + selective capture shell + conditioning/wash state`

## Suggested First Prototype Windows

Start with these bounded windows:

- particle diameter: `300-700 um`
- shell thickness: `5-25 um`
- dry media target: one small bench batch only
- no more than `3` shell-loading variants in the first pass

## Suggested Build Logic

### Step 1: Core Selection

Choose one mechanically robust bead or granule scaffold that is:

- water-stable
- easy to wash and size
- tolerant to a later shelling or coating step

Good first-pass logic:

- prioritize packing stability over maximum chemistry complexity
- reject very fine powders for the first PF1-C1 build

### Step 2: Surface Activation

Prepare the outer surface so that a selective shell can be anchored reproducibly.

The first pass should optimize for:

- stable shell attachment
- limited pore blockage
- limited pressure penalty

### Step 3: Selective Shell Formation

Build a thin shell rather than a fully transformed bulk particle.

The shell should be designed to balance:

- PFAS affinity
- water compatibility
- regeneration survival
- acceptable hydraulic behavior

For the first pass, keep shell loading modest rather than maximal.

### Step 4: Conditioning

After shell formation:

- wash to remove loosely bound residues
- condition into a reproducible wet state
- record wet mass, dry mass, and particle-size distribution

### Step 5: First Bounded Replay

Run one immediate replay build with the same profile.

If the second build cannot reproduce the first shelling result, stop and retune before broader screening.

## First Experimental Variant Set

The first bounded variant set should be small:

1. low shell-loading variant
2. mid shell-loading variant
3. high shell-loading variant

Do not open more variables until one shell-loading window behaves coherently.

## Minimum Characterization Before PFAS Screening

Before any meaningful PFAS capture screen, record:

- particle-size band
- visual integrity after wash
- approximate shell fraction or shell-growth indicator
- dry/wet mass relationship
- simple pressure-drop comparison versus an untreated scaffold

## Early Kill Conditions

Kill PF1-C1 early if any of these appear:

- clear particle breakdown under wash or conditioning
- large pressure penalty before useful capture appears
- visible shell shedding
- regeneration collapse after a short first replay

## Early Continue Conditions

Continue PF1-C1 if:

- shell formation is reproducible
- particle handling remains practical
- pressure behavior stays within a bounded usable range
- early regeneration does not collapse immediately

## What This SOP Is Not

- not a validated industrial manufacturing route
- not a certified product process
- not a final precursor list
- not a scale-up instruction

It is simply the cleanest current path for turning PF-1 into a first real screening material.
