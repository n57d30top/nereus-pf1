# PF-1 Claim Guardrails

- `PF-1` as a bounded closed-loop PFAS polishing architecture
- methodology and benchmark-framing disclosure
- bounded internal surrogate benchmark summary
- explicit statement that the package is technical and sanitized

- `PF-1 is x% better than market filters`

- pilot, field, wet-lab, household, or public-water treatment claims
- lifecycle-economics promises
- universal `Nereus` family claims

Use `PF-1` only as:

`a private technical package for a bounded closed-loop PFAS polishing module with a sanitized benchmark and methodology dossier`
